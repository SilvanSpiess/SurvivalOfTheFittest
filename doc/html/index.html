  • #Heading intro_sec Introduction
  • This project is a C++ simulator that models the interaction of different life forms in a fictional world. The world contains instances of an abstract class Living,
  • which serves as the base class for all creatures and nature elements within this world. The following living things exist in this fictional world:
  • * List item 1 Empty
  • * List item 2 Grass
  • * List item 3 Rabbits
  • * List item 4 Foxes
  • Every form of life multiplies according to certain biological patterns. The individual life forms are either predator or prey to another life form. However, there are
  • extensions possible which make the simulation more realistic. One such extension which applies to foxes and rabbits are diseases, which break out at random. There are
  • other creatures possible besides the species foxes and rabbits, such as owls and mice.
  • The visualization heavily depends on your personal preferences. It will be a command-line interface (CLI) visualization. However, additional control mechanisms can be
  • implemented. Furthermore, the population statistics as well as the simulation properties should be visible.
  • #Heading install_sec Installation
  • ##Heading; step1 Step 1: Clone
  • Clone the repository:
  • *git clone git@g.nosp@m.itla.nosp@m.b.ti..nosp@m.bfh..nosp@m.ch:btf2232/students_fs23/project/wotf-grp-09.git*
  • ##Heading 2; step2 Step 2: Build
  • Go into the project directory: cd world-of-fittest-simulation
  • Create a build directory: mkdir build && cd build
  • Run CMake to generate the build files: cmake ..
  • Build the application: make
  • # usage_sec Usage
  • Run the application: *./world-of-fittest-simulation*
  • Follow the CLI prompts to configure the simulation.
  • View the simulation statistics and properties at the end of the simulation.
